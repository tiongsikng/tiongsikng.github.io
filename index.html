    Malaysian Digital Signature Algorithm Proposal: TNC Signature Scheme (MySEAL)

    Proposed the TNC Signature, compatible with Schnorr signatures, providing full security proof and implementation using C and Libgcrypt for both discrete logarithm and elliptic curve settings. Schnorr 서명과 호환되는 TNC 서명을 제안하였고, 이산 로그 및 타원 곡선 설정 모두에서 C 및 Libgcrypt를 사용하여 완전한 보안 증명과 구현을 제공함. Schnorr署名と互換性のあるTNC署名を提案し、離散対数および楕円曲線設定の両方でCおよびLibgcryptを使用した完全なセキュリティ証明と実装を提供。 提出了TNC签名，与Schnorr签名兼容，使用C和Libgcrypt在离散对数和椭圆曲线设置中提供了完整的安全证明和实现。
